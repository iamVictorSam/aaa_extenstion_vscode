# Dart Test Snippet

## Purpose

This extension provides a code snippet to quickly generate a basic test case template in Dart files.

## Usage

- Type aaa.

- Select the "aaa" suggestion.

Generated Code:

```dart
Dart
test('should ', () async {
  //arrange
  
  //act
  
  //assert
  
});
```
